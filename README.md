# ローズ研修デッキ ― デザイン再現プロンプト

「環境分析フレームワーク入門」で使った、上品ローズ基調の法人研修デザインを、別テーマでも再現するためのプロンプト一式です。
用途に合わせて **A. コピペ用（短縮版）** か **B. 詳細仕様** を使ってください。

---

## A. コピペ用プロンプト（短縮版）

> あなたは一流のエンジニア兼デザイナーです。pptxgenjs で 16:9ワイド（13.333 × 7.5 inch）の法人研修用スライドを作ってください。テーマは【ここに内容】。個人名・メール・地名など個人情報は出さず、登場人物は役割名のみ。
>
> **配色（上品ローズ／必ずこのhexを使用・`#`は付けない）**
> - ワインローズ `5C2438`（タイトル／まとめ背景・強調カード・見出し）
> - ディープワイン `47172C`（背景の重なり装飾）
> - ローズ `A23E5C`（主アクセント・キッカー・番号丸）
> - ミッドローズ `BE647D`／ダスティローズ `CE8A99`（矢印・補助線・ページ番号）
> - ブラッシュ `EAD0D6`／ライトブラッシュ `F7ECEF`（アイコン丸・淡カード地）
> - 白 `FFFFFF`（本文スライド地）／インク `3D2A30`（本文濃色）／ミュート `8C7077`（補足）
> - ローズゴールド `B98C63`（タグライン等にごく少量）
>
> **フォント**：見出し・大きな数字＝Noto Serif CJK JP、本文・ラベル＝Noto Sans CJK JP。
>
> **共通モチーフ**：アイコンは必ず「ライトブラッシュの丸（ブラッシュ1ptの輪郭）」の中に置き、全スライドで反復。番号はワインの丸＋白いセリフ数字。
>
> **レイアウト規則**：本文スライドは左上に〈アイコン丸→英語キッカー（11pt ROSE 太字・字間広め）→セリフ見出し27pt〉。カードは角丸長方形＋白地＋ブラッシュ枠＋淡いドロップシャドウ。強調カードのみワイン地。フッターは左に「うさうさ研修工房」、右に2桁ページ番号。
>
> **構成は WHY → WHAT → HOW**。説明は「定義→たとえ→詳細→一言まとめ」の4段で、文系・非エンジニアにも伝わる平易さ。フォーマルなので装飾絵文字は使わない。
>
> **禁止**：見出し下線、アクセントの帯／サイドストライプ、本文の中央寄せ、クリーム/ベージュ地、全色を等価に使うこと、テキストのはみ出し。

## B. 詳細仕様（ブランド・スタイルガイド）

### 1. キャンバス
| 項目 | 値 |
|---|---|
| 比率 | 16:9 ワイド |
| サイズ | 13.333 × 7.5 inch（pptxgenjs の `defineLayout`） |
| 外余白 | 上下左右 0.7 inch（最低 0.5） |
| カード間の間隔 | 0.27〜0.3 inch（混在させない） |

### 2. カラーパレット（`#` を付けない6桁hex）
| 役割 | 名称 | hex | 使いどころ |
|---|---|---|---|
| 主・濃 | ワインローズ | `5C2438` | 表紙/まとめ背景、強調カード、見出し文字 |
| 主・最濃 | ディープワイン | `47172C` | 背景の同心円・重なり装飾、丸の影 |
| アクセント | ローズ | `A23E5C` | キッカー、番号丸、矢印元、「→」 |
| 補助 | ミッドローズ | `BE647D` | 矢印、枠線（半透明）、装飾線 |
| 補助 | ダスティローズ | `CE8A99` | ページ番号、サブ装飾 |
| 淡 | ブラッシュ | `EAD0D6` | アイコン丸の輪郭、淡カード地 |
| 淡・最淡 | ライトブラッシュ | `F7ECEF` | アイコン丸の地、淡パネル |
| 地 | 白 | `FFFFFF` | 本文スライド背景、カード地 |
| 文字・濃 | インク | `3D2A30` | 本文の見出し・強調語（黒の代わり） |
| 文字・淡 | ミュート | `8C7077` | 補足・キャプション |
| 差し色 | ローズゴールド | `B98C63` | タグライン等にごく少量のみ |

**ドミナンス**：表紙・まとめは `5C2438` を画面の6〜7割に。本文スライドは白地を主役にし、ローズはアクセントとして1〜2割。全色を等価に並べない。

### 3. タイポグラフィ
- 見出し・大きな数字 = **Noto Serif CJK JP**（明朝の上品さ）
- 本文・ラベル・英語キッカー = **Noto Sans CJK JP**
- 置き換え時の近似：セリフ→游明朝／Hiragino Mincho、サンセリフ→游ゴシック／Hiragino Kaku Gothic

| 要素 | サイズ目安 |
|---|---|
| 表紙タイトル | 44pt セリフ太字 |
| 本文タイトル | 27pt セリフ太字 |
| キッカー（英語ラベル） | 11〜13pt サンセリフ太字・字間+2・大文字 |
| セクション見出し | 14〜16pt サンセリフ太字 |
| 本文 | 11.5〜13pt |
| キャプション/フッター | 9.5〜10pt（ミュート色） |
| 番号（丸の中） | 20〜34pt セリフ |

### 4. 共通モチーフ（全スライドで反復）
1. **アイコン丸**：`F7ECEF` 塗り＋`EAD0D6` 1pt 輪郭の正円に、ローズ（`A23E5C`）または白のアイコンを中央配置。サイズは丸0.66〜0.95 inch。
2. **番号丸**：`5C2438` の正円＋白いセリフ数字。
3. アイコンは react-icons を 256px 以上でPNGラスタライズして挿入。

### 5. スライド・パターン
**本文スライド（共通ヘッダー）**
- 左上 0.7,0.55 にアイコン丸（0.78）
- その右に 英語キッカー（11pt `A23E5C` 太字・字間2）
- 直下にセリフ見出し 27pt `3D2A30`
- 見出し下にリード文 14pt `8C7077`（1〜2行）

**カード**
- `ROUNDED_RECTANGLE`、角丸 `rectRadius` 0.08〜0.1
- 通常：白地＋`EAD0D6` 1.25pt 枠＋淡い影（outer / `000000` / blur 7 / offset 2 / angle 90 / opacity 0.10）
- 強調：`5C2438` 地＋やや濃い影（color `5C2438` / blur 9 / opacity 0.16）

**フッター**：左下に「うさうさ研修工房」（9pt `8C7077`）、右下に2桁ページ番号（10pt `CE8A99`）。

**表紙／まとめ**：`5C2438` 全面。右上または左下に同心円（`47172C`＋半透明ローズ枠）で奥行きを出す。タグライン「面白きこともなき世を面白く」をローズゴールドの斜体で角丸チップに。

### 6. レイアウトのバリエーション（毎回変える）
- 4枚カード横並び（アジェンダ／PEST／まとめ）
- 左ステートメント（濃色）＋右3行リスト（WHY）
- 2カラム対比（マクロ↔ミクロ、定量↔定性）
- 2×2マトリクス＋行ラベル（外部/内部）
- 中央＋四方向の矢印図（ファイブフォース）
- 表（1PL〜4PL早見表。該当セルを `EAD0D6` でハイライト）

### 7. コンテンツの作法（うさうさ研修工房スタイル）
- 構成は **WHY → WHAT → HOW** の順。
- 各概念は **定義 → たとえ → 詳細 → 一言まとめ** の4段で。
- 文系・非エンジニアにも伝わる平易な言葉。必要なら身近なたとえ（例：うさうさラーメン店）。
- 各フレームワークに必ず「業界での具体例」を添える。
- フォーマル文脈では装飾絵文字を使わない。結論先出し・事実と所感を分ける。
- 個人情報は出さない：人名・メール・地名は除外し、登場人物は役割名（荷主／物流事業者 等）のみ。

### 8. 禁止事項（AIっぽさ・崩れの回避）
- 見出しの下線、アクセントの帯・サイドストライプ、カード縁の細い色帯
- 本文の中央寄せ（タイトルのみ中央可）
- クリーム/ベージュ地（`F5F5DC` `FAF0E6` 等）
- 全色を等しい面積で使う／青系デフォルト
- テキストのはみ出し（収まらなければ縮小・分割・箱拡大）
- hex に `#` を付ける、8桁hexで透明度を表す（影は `opacity` で指定）

---

### 使い方メモ
- 【ここに内容】にテーマ（例：「SWOT分析」「ロジカルシンキング入門」）を入れて A をそのまま投げれば同じ世界観で再現できます。
- 別ブランド色にしたい時は §2 の hex を差し替えるだけで全体トーンが変わります（モチーフ・レイアウト規則はそのまま流用可）。
